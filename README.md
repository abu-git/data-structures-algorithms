# data-structures-algorithms

This repo contains python files while practicing lessons on a couple of concepts from Data Structures and Algorithms.


## 📦 Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/abu-git/data-structures-algorithms.git
cd data-structures-algorithms
```

## 🛠 Usage

Notice that there are various python files and I am assuming you are making use of an IDE like Visual Studio Code.
You can run any file on terminal


```bash
python brackets.py
```

Repeat for any file in the directory. For example:

```bash
python binarySearch.py
```

or 

```bash
python binarySearchAlgorithm.py
```


## Python3 terminal users
I am also aware that some users might need to run **python3** and not **python**


```bash
python3 brackets.py
```

Repeat for any file in the directory. For example:

```bash
python3 binarySearch.py
```

or 

```bash
python3 binarySearchAlgorithm.py
```
